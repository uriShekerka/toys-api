Toys api documentation

Routes

  • Get 10 products by page number:/toys/?perPage=10 (GET)
  • Get X products in each page:/toys/?perPage=10&page=2 (GET)
  • Search toy by name or info:/toys/search/?s= name or info (GET)
  • Get all products by category:/toys/category/:category-name (GET)

    • Get toy - (only for register user - must send user's auth-token through headers):/toys (POST)
  • name (required)
  • info (required)
  • category (required)
  • img_url
  • price (required)
  • user_id (required)

    • Edit product - (only for register user - must send user's auth-token through headers):/toys/:_id (toy) (PUT)
  • name (required)
  • info (required)
  • category (required)
  • img_url
  • price (required)
  • user_id (required)

    • Delete product - (only for register user - must send user's auth-token through headers):/toys/:_id (toy) (DELETE)
  • name (required)
  • info (required)
  • category (required)
  • img_url
